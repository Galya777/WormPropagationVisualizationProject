CodeRed

Vulnerability that exploits: IIS Buffer Overflow – Уязвимост в Microsoft IIS 5.0 (WebDAV), позволяваща записване на shellcode чрез HTTP GET заявка.
CVE: CVE-2001-0500

Ways of attacking:Използва специално създаден HTTP GET с препълване на буфера, насочен към порт 80 на IIS уеб сървър.
Експлойтът записва код в паметта и го изпълнява.

Reproducing: След заразяване, сканира произволни IP адреси за други машини с отворен порт 80.
Автоматично разпространява себе си към други уязвими хостове.
Не изисква файлове — работи изцяло в паметта.

Payload behavior: Промяна на уеб страницата с „Hacked by Chinese!“.
Flood атака срещу whitehouse.gov (DoS).
Различни фази: 1. Заразяване, 2. Размножаване, 3. Атака.
